# 编程世界全貌 | Programming World Landscape

> 面向编程小白和 aspiring indie developers 的完整指南

[![GitHub](https://img.shields.io/github/stars/Leochenrw/coder-landscape)](https://github.com/Leochenrw/coder-landscape)

## 项目简介

这是一个涵盖 **66个编程方向** 的全面指南，从基础开发到前沿技术，帮助你：

- 了解每个方向是做什么的
- 掌握 AI 时代的核心技能
- 找到变现路径和入门方法

## 双版本输出

| 版本 | 文件 | 特点 |
|------|------|------|
| **网页版** | `coder-landscape.html` | 交互式导航、进度追踪、AI搜索、响应式设计 |
| **文档版** | `coder-landscape-guide.docx` | A4横版、两栏布局、适合打印 |

## 快速开始

```bash
# 安装依赖
npm install

# 生成本地预览
npx serve coder-landscape.html

# 生成 Word 文档
node generate-coder-landscape.js
```

## 66章完整目录

### 基础开发（1-10）
前端开发、移动端开发、后端开发、数据库开发、全栈开发、桌面应用开发、小程序开发、低代码/无代码、API开发、DevOps

### 游戏与娱乐（11-16）
游戏开发、抖音小游戏、独立游戏、游戏MOD开发、电竞技术、元宇宙开发

### 人工智能（17-24）
机器学习、深度学习、NLP、计算机视觉、推荐系统、具身智能、AI应用开发、大模型微调

### 数据与商业（25-32）
数据分析、商业智能、数据工程、量化交易、增长黑客、产品管理、技术写作、技术咨询

### 安全与系统（33-38）
网络安全、渗透测试、区块链开发、智能合约、Web3开发、密码学

### 云计算与基础设施（39-45）
云计算、云原生、容器技术、Kubernetes、Serverless、边缘计算、物联网

### 内容创作与媒体（46-52）
技术视频、技术播客、直播技术、音视频开发、3D建模与渲染、数字人、AI绘画

### 自动化与工具（53-58）
爬虫开发、自动化测试、RPA机器人、浏览器插件、VSCode插件开发、CLI工具开发

### 新兴与前沿（59-66）
机器人编程、无人机开发、嵌入式开发、自动驾驶、AR/VR开发、空间计算、脑机接口、量子计算

## 每章12个板块

1. **这是什么** - 专业解释 + 大白话 + 2026现状
2. **做什么** - 工作内容 bullet list
3. **细分有哪些** - 子方向表格
4. **核心工具/语言** - 工具列表 + AI原生版本
5. **AI 工具清单** - 工具表格
6. **最小可行知识** - ✅必须掌握 vs ❌不需要
7. **变现路径** - 方式、收入、可行性分析
8. **AI 冲击评估** - 替代风险/增强程度/新机会
9. **避坑清单** - 致命级/重伤级/中伤级
10. **掌握路径** - AI时代学习路线
11. **中文最佳资源** - 首推/在线/进阶
12. **如何入门** - 4步入门行动清单

## 技术栈

- **HTML**: 单页交互应用
- **docx.js**: Word 文档生成
- **LocalStorage**: 阅读进度、收藏功能
- **Claude API**: AI 搜索功能
